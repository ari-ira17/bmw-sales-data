# BMW sales data 2010-2024

## Описание датасета

Данный датасет содержит информацию о продажах автомобилей **BMW** по различным регионам мира. Каждая строка представляет собой запись о модели автомобиля с указанием её характеристик, цены и объёма продаж.

---

### Структура данных

| Колонка | Описание | Тип |
|--------|--------|------|
| `Model` | Модель автомобиля (например, `3 Series`, `X3`, `i8`, `M5`, `7 Series`) | Категориальная |
| `Year` | Год выпуска автомобиля | Числовая (год) |
| `Region` | Регион продажи (`Asia`, `Europe`, `North America`, `South America`, `Middle East`) | Категориальная |
| `Color` | Цвет автомобиля (`Red`, `Blue`, `Black`, `White`, `Silver` и др.) | Категориальная |
| `Fuel_Type` | Тип топлива (`Petrol`, `Diesel`, `Hybrid`, `Electric`) | Категориальная |
| `Transmission` | Тип коробки передач (`Manual`, `Automatic`) | Категориальная |
| `Engine_Size_L` | Объём двигателя, литры | Числовая |
| `Mileage_KM` | Пробег, километры | Числовая |
| `Price_USD` | Цена автомобиля в долларах США | Числовая |
| `Sales_Volume` | Объём продаж (количество проданных единиц данной конфигурации) | Числовая |

---

### Пример записи

```
Model: 5 Series  
Year: 2016  
Region: Asia  
Color: Red  
Fuel_Type: Petrol  
Transmission: Manual  
Engine_Size_L: 3.5  
Mileage_KM: 151748  
Price_USD: 98740  
Sales_Volume: 8300
```

---

### Назначение

Датасет предназначен для:
- Анализа взаимосвязей между характеристиками автомобилей и их ценой,
- Изучения предпочтений покупателей по регионам и цветам,
- Построения визуализаций (гистограммы, тепловые карты, box plots, countplot и др.),
- Подготовки данных для задач машинного обучения (после кодирования категориальных признаков).
