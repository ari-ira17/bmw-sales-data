# BMW sales data 2010-2024

## Описание датасета

Данный датасет содержит информацию о продажах автомобилей **BMW** по различным регионам мира. Каждая строка представляет собой запись о модели автомобиля с указанием её характеристик, цены и объёма продаж.

---

## Структура данных

| Колонка | Описание | Тип |
|--------|--------|------|
| `Model` | Модель автомобиля (например, `3 Series`, `X3`, `i8`, `M5`, `7 Series`) | Категориальная |
| `Year` | Год выпуска автомобиля | Числовая |
| `Region` | Регион продажи (`Asia`, `Europe`, `North America`, `South America`, `Middle East`) | Категориальная |
| `Color` | Цвет автомобиля (`Red`, `Blue`, `Black`, `White`, `Silver` и др.) | Категориальная |
| `Fuel_Type` | Тип топлива (`Petrol`, `Diesel`, `Hybrid`, `Electric`) | Категориальная |
| `Transmission` | Тип коробки передач (`Manual`, `Automatic`) | Категориальная |
| `Engine_Size_L` | Объём двигателя, литры | Числовая |
| `Mileage_KM` | Пробег, километры | Числовая |
| `Price_USD` | Цена автомобиля в долларах США | Числовая |
| `Sales_Volume` | Объём продаж (количество проданных единиц данной конфигурации) | Числовая |

---

## Пример записи

```
Model: 5 Series  
Year: 2016  
Region: Asia  
Color: Red  
Fuel_Type: Petrol  
Transmission: Manual  
Engine_Size_L: 3.5  
Mileage_KM: 151748  
Price_USD: 98740  
Sales_Volume: 8300
```

---

## Примеры использования

- **Анализ цен**: как `Price_USD` зависит от года выпуска, пробега и объёма двигателя.  
- **Региональные различия**: сравнение средних цен и объёмов продаж (`Sales_Volume`) по регионам.  
- **Популярные конфигурации**: какие модели и цвета (`Model`, `Color`) чаще всего выбирают.  
- **Визуализация**: гистограммы, box plots, тепловые карты корреляций.  
- **Подготовка данных**: one-hot кодирование категориальных признаков для ML.  
- **Статистика**: проверка нормальности распределения (например, для `Price_USD`).

## Технические детали

- **Язык программирования**: Python 3  
- **Основные библиотеки**:
  - `pandas` — обработка и анализа данных

  - `numpy` — работа с числовыми массивами
- **Формат данных**: CSV 
- **Окружение**: виртуальное окружение (`venv`)
